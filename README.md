# FancyForm

Build v0.2.5  

For more info on using Fancy Form, see the docs: http://www.fancyjs.com/docs.html


##Build

```sh
$ npm install
```
  
Generate dev version
```sh
$ grunt dev
```

Generate prod version
```sh
$ grunt prod
```

Generate full built version
```sh
$ grunt full
```

Build files are under `fancyform` directory
